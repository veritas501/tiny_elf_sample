# Tiny ELF sample

## pop_shell_x64

- arch: x64
- shellcode pos: 0x78
- size: 180 bytes

just pop a shell.

test:

```
./pop_shell_x64
$ whoami
veritas
$
```



## pop_shell_preload_x64.so

- arch: x64
- shellcode pos: 0x170
- size: 428 bytes

just pop a shell, but can be used to hijack program using LD_PRELOAD or dlopen.

test:

```
./pop_shell_preload_x64.so
$ whoami
veritas
$

LD_PRELOAD=./pop_shell_preload_x64.so /usr/bin/true
$ whoami
veritas
$
```
